
## About 
Simple chat realtime


#### Installation
copy and paste in your terminal

```bash
git clone https://github.com/MRizki28/Chat-App.git
```

install composer

```bash
composer install
```

install node module

```bash
npm install
```

generate key

```bash
php artisan key:generate
```

### Migration table

```bash
php artisan migrate::fresh
```

### Run Project

```bash
php artisan serve
```

```bash
php artisan reverb:start
```

```bash
npm run dev
```
